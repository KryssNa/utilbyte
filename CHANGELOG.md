# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2024-01-01

### Added

- Initial release with 45+ browser-based tools
- **Image Tools**: Compress, Crop, Resize, Format Convert, Background Remove, Blur, OCR
- **PDF Tools**: Merge, Split, Compress, Rotate, Edit, PDF-to-Image, Image-to-PDF
- **Text Tools**: Word Counter, Case Converter, Lorem Ipsum, Text Formatter, Remove Duplicates
- **Developer Tools**: JSON Formatter, Base64, JWT Decoder, Regex Tester, Hash Generator, Cron Parser, UUID Generator, URL Encoder, Markdown Renderer, Diff Checker, Code Beautifier, SQL Formatter, WebSocket Client, Online Compiler, API Client, Request Catcher
- **Utility Tools**: QR Code, Barcode, Password Generator, Color Converter, Unit Converter, Timestamp, Countdown
- **Video Tools**: Compress Video, Video to Audio, Video to GIF
- Dark and light theme support
- Responsive design for all screen sizes
- SEO optimization with structured data, sitemaps, and meta tags
- Privacy-first architecture — all processing runs client-side
